# 📊 Data Analytics Internship Projects

## 📌 Overview

This repository showcases the projects completed as part of my **Data Analytics Internship**. The tasks focus on preparing, exploring, and analyzing an e-commerce dataset using Python. Each project demonstrates practical data analytics techniques, including data cleaning, preprocessing, statistical analysis, and data visualization.

---

## 🎯 Internship Tasks

### Week 1 – Data Cleaning and Preprocessing

The objective of this task was to improve the quality of the raw dataset by identifying and correcting data issues before analysis.

**Tasks Performed:**
- Identified missing values
- Handled missing data
- Checked and removed duplicate records
- Verified and corrected data types
- Prepared the dataset for analysis
- Exported the cleaned dataset

---

### Week 2 – Exploratory Data Analysis (EDA)

The objective of this task was to explore the cleaned dataset, identify patterns, detect outliers, and generate meaningful business insights using statistical analysis and visualizations.

**Tasks Performed:**
- Dataset overview
- Descriptive statistics
- Distribution analysis
- Payment method analysis
- Order status analysis
- Referral source analysis
- Outlier detection
- Correlation analysis
- Key business insights
